## 🔍 About the Application

The-agent-sandbox-taxonomy helps you understand and score AI agent sandboxes. It uses seven defense layers, seven threat categories, and three evaluation dimensions to rate over 20 different sandbox setups. This framework shows how AI environments keep your data and systems safe. You do not need any technical background to use it.

This app gives you a clear view of sandbox security. It works on Windows computers and uses simple tools to help you study or compare sandboxes.
